# Smart File Organizer (Python)

## Overview

This project is a Python script that automatically organizes files in a directory into folders based on file type.

## Features

* Sorts files into categories (Documents, Images, Videos, Others)
* Automatically creates folders
* Works on any specified directory

## Technologies Used

* Python
* os module
* shutil module

## How to Run

1. Clone the repository
2. Update the folder_path in organizer.py
3. Run:
   python organizer.py
